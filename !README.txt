Protoze muj program je rozdelen do nekolika ruznych slozek, ke spocitani radku pouzijte prikaz:

Windows:
	cloc Angular\Meteo\src MeteoStationListener\src Simulator spring\src --exclude-lang="HTML,CSS,Properties"
Linux:
	cloc Angular/Meteo/src MeteoStationListener/src Simulator spring/src --exclude-lang="HTML,CSS,Properties"


- Toto je pouze pro vase pohodli, aby jste nemusel, hledat jake slozky pocitat a jake ne, ale klidne si to overte.
